 📊 Financial Portfolio Analytics -- Power BI Project

 📌 Sobre o Projeto

Este projeto consiste em um dashboard financeiro desenvolvido no Power
BI com o objetivo de analisar a performance de uma carteira de
investimentos fictícia.

O foco principal é demonstrar conhecimentos em:

-   Modelagem de dados (Modelo Estrela)
-   Criação de métricas em DAX
-   Processo de ETL no Power Query
-   Construção de dashboard com visual profissional

------------------------------------------------------------------------

 🎯 Objetivo

Criar um painel interativo que permita acompanhar:

-   Patrimônio total
-   Dividendos acumulados
-   Rentabilidade da carteira
-   Distribuição por setor
-   Diversificação entre FII e Ações

------------------------------------------------------------------------

 🛠️ Ferramentas Utilizadas

-   Power BI
-   Power Query
-   DAX
-   Excel (fonte de dados)

------------------------------------------------------------------------

 🧠 Modelagem de Dados

O projeto foi estruturado utilizando Modelo Estrela (Star Schema).

 🔹 Tabelas Fato

-   fAportes
-   fDividendos

 🔹 Tabelas Dimensão

-   dCalendario
-   dAtivos
-   dVisaoGeral

Relacionamentos configurados como Muitos-para-Um, garantindo
organização e melhor performance nas análises.

------------------------------------------------------------------------

 📊 Principais Métricas Criadas

-   Patrimônio Total
-   Dividendos Acumulados
-   Rentabilidade (%)
-   Valorização da Cota (%)
-   Preço Médio
-   Preço Atual
-   TIR (calculada no Excel)

Exemplo de cálculo de Rentabilidade:

Rentabilidade % = (Valor Total + Dividendos - Valor Aplicado) / Valor
Aplicado

------------------------------------------------------------------------

 📈 Visualizações do Dashboard

-   Evolução de Dividendos (gráfico de colunas)
-   Alocação por Setor (barras horizontais)
-   Diversificação FII vs Ação (gráfico de rosca)
-   Distribuição detalhada por Ativo (tabela)
